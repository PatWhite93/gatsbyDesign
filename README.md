# Slick's Slices (Tutorial)
###Using Gatsby and React

## General info
A pizzaria homepage, where customers can learn about the staff, and order pizzas.  Uses Gatsby 
	
## Technologies
Project is created with:
* Gatsby
* React
* Sanity
* Styled Components
